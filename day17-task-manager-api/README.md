# Task Manager API

[Live: https://task-manager-api.onrender.com](https://backend-learning-onvp.onrender.com)

## Auth Routes
- POST /auth/register
- POST /auth/login

## Task Routes
- GET /tasks
- POST /tasks
- PUT /tasks/:id
- DELETE /tasks/:id

## Features
- JWT Authentication
- User-task ownership
- Query filtering
- Security middleware (Helmet, CORS, XSS-clean, etc.)
- Deployed on Render
- MongoDB Atlas database
